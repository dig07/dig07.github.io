GW150914 LISA to A+ LIGO

Visualisation of a GW150914-like source, initially 3.2 years from merger, transiting from the LISA band to the A+ LIGO band. Created using the amazing Manim Python library.

LISA stellar mass-binary tracks

Visualisation of a set of stellar mass binaries, chirping through the LISA frequency band. Also includes a model of how the confusion noise changes as a function of time.

Python library to create these animations with other source types coming soon!

Semi-coherent search

Visualisation of the semi-coherent phase maximisation applied to a time domain sinusoidal waveform. Animated version of figure 1 in this paper.

Hierarchical PSO

Visualisation of a hierarchical Particle Swarm Optimizer, proposed as a search algorithm for sorting through the EMRI likelihood peak forest. Implemented in this github library

LISA Orbits
